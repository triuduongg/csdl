# Bài tập lớn môn Cơ sở dữ liệu
## Cách cài đặt:
### Tải xuống xampp
### Tải xuống thư mục dự án:
#### Di chuyển thư mục dự án đến thư mục htdocs của xampp
#### Chạy xampp:
##### Tạo cơ sở dữ liệu tên là csdl
##### Import file csdl.sql trong thư mục uploads của dự án
## Cách chạy:
### Chạy xampp:
#### Bật Apache và MySQL
#### Truy cập đường dẫn https://localhost/csdl
